La Fabrique

Avant de lancer la Fabrique, activez le javascript sur votre navigateur.

A quoi sert la Fabrique ?

Il s'agit d'une application internet pour construire facilement des exercices de mathématiques en ligne qui exploitent le moteur de calcul formel XCAS . Vous n'aurez que quelques lignes de code à taper pour chaque exercice.

Que vous faut-il ?

La fabrique fonctionne sous les navigateurs suivants :

  • Firefox (≥2)
  • Safari
  • Chrome (≥ 11)
  • Internet Explorer (≥ 9, pas sous les versions antérieures !)

Il est conseillé de télécharger et installer XCAS .
